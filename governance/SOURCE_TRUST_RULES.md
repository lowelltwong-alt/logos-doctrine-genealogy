# Source Trust Rules

No source rows, source text imports, source corpora, commentary corpora, or
Scripture text are authorized in the scaffold.

Future source intake must record:

- source identity;
- source owner repo;
- licensing status;
- trust tier;
- citation locator;
- source scope;
- method of use;
- review status;
- provenance;
- reviewer.

Metadata about a source is not source text permission. A useful source is not
automatically an authoritative source.
